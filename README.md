# CopyMaster - 视频文案提取神器

一键提取B站、小红书、抖音等主流平台的视频文案，支持AI智能摘要和文案优化。

## 本项目由[阿里云ESA](https://www.aliyun.com/product/esa)提供加速、计算和保护

![阿里云ESA](https://img.alicdn.com/imgextra/i3/O1CN01H1UU3i1Cti9lYtFrs_!!6000000000139-2-tps-7534-844.png)

## 功能特性

- **多平台支持**：B站、小红书、抖音、快手、微博、YouTube
- **智能提取**：自动识别平台，一键提取视频文案/字幕
- **AI增强**：集成通义千问，自动生成摘要和关键词
- **文案优化**：支持多种风格的AI文案优化
- **历史记录**：本地保存提取历史，随时查看
- **一键操作**：复制、下载，快速获取文案

## 技术栈

- **前端**：React 18 + TypeScript + Tailwind CSS + Framer Motion
- **后端**：ESA Edge Functions (边缘函数)
- **AI**：通义千问 API
- **存储**：ESA Edge KV + 边缘缓存

## How We Use Edge

本项目充分利用了阿里云ESA的边缘计算能力：

### 1. 边缘函数 (Edge Functions)

- **视频信息提取**：在边缘节点直接调用各平台API，减少回源延迟
- **AI处理**：边缘节点调用千问API，就近处理用户请求
- **请求代理**：边缘函数作为代理层，解决跨域问题

### 2. 边缘缓存 (Edge Cache)

- **提取结果缓存**：相同视频链接的提取结果缓存1小时
- **减少重复请求**：避免对源站API的重复调用
- **提升响应速度**：缓存命中时毫秒级响应

### 3. 边缘KV存储 (Edge KV)

- **持久化存储**：提取结果存储24小时
- **跨节点共享**：全球边缘节点共享数据
- **高可用保障**：数据多副本存储

### 4. 全球加速

- **静态资源CDN**：前端资源全球分发
- **智能路由**：自动选择最优边缘节点
- **低延迟访问**：全球用户就近访问

## 使用说明

1. **粘贴链接**：将视频分享链接粘贴到输入框
2. **自动识别**：系统自动识别平台类型
3. **提取文案**：点击提取按钮获取文案
4. **AI增强**（可选）：
   - 在设置中配置千问API Key
   - 自动生成摘要和关键词
   - 使用AI优化功能改写文案
5. **导出使用**：复制或下载提取的文案

## 配置千问API

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册/登录账号
3. 创建API Key
4. 在设置中粘贴API Key

配置后可使用：
- AI自动摘要
- 关键词提取
- 文案优化（正式/口语/营销/精简）
